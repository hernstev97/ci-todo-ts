# ci-todo

A simple todo app written in typescript that works within the local storage.

## Features
1. edit
2. add
3. delete
4. done-state
5. sort via arrow buttons
6. sort via drag & drop
7. save data in local storage

## Technical requirements
* Responsive
    * mobile: 360px
    * tablet: 720px
    * desktop: 1280px
* use rem
    * mobile: 16px
    * Tablet: 18px
    * desktop: 20px
* no jQuery
* no css frameworks
* no js/ts-libraries
* use webpack
* use postcss
* use plain js or ts
* ?test with jest
* compatibility: most recent browsers (Chrome, Firefox, Safari und ggf. Edge)
